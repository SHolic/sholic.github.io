模型原理

CRF与HMM区别

  • HMM不知道隐藏状态，CRF知道隐藏状态

  • HMM隐藏状态只依赖前一个隐藏状态，CRF隐藏状态可以依赖上下文（Ngram），linear_CRF依赖前一个隐藏状态

  • HMM的隐藏状态转移概率是规范化的（和等于1），CRF隐藏状态转移概率是非规范化的（权重乘以1或0）

模型原理

HMM模型特征

  1. 问题是基于序列的，比如时间序列，或者状态序列。
  2. 问题中有两类数据，一类序列数据是可以观测到的，即观测序列；而另一类数据是不能观察到的，即隐藏状态序列，简称状态序列。

HMM定义

假设$Q=\lbrace q_1,q_2,\ldots,q_N\rbrace$是所有可能的隐藏状态的集合，$V=\lbrace v_1,v_2,\ldots,v_M\rbrace$ 是所有可能的观测状态的集合

对于一个长度为$T$的序列，$I=\lbrace i_1,i_2,\ldots,i_T\rbrace$是对应的状态序列，$O=\lbrace o_1,o_2,\ldots,o_T\rbrace$是对应的观察序列

观测独立性假设。即任意时刻的观察状态只仅仅依赖于当前时刻的隐藏状态

在时刻$t=1$的隐藏状态概率分布$\Pi$

三个问题

  1. 评估观察序列概率。即给定模型$\lambda=(A,B,\Pi)$和观测序列$O=\lbrace o_1,o_2,\ldots,o_T\rbrace$，计算在模型$\lambda$下观测序列$O$出现的概率$P(O|\lambda)$。这个问题的求解需要用到前向后向算法。这个问题是HMM模型三个问题中最简单的。
  2. 模型参数学习问题。即给定观测序列$O=\lbrace o_1,o_2,\ldots,o_T\rbrace$，估计模型$\lambda=(A,B,\Pi)$的参数，使该模型下观测序列的条件概率$P(O|\lambda)$最大。这个问题的求解需要用到基于EM算法的鲍姆-韦尔奇算法。这个问题是HMM模型三个问题中最复杂的。
  3. 预测问题，也称为解码问题。即给定模型$\lambda=(A,B,\Pi)$和观测序列$O=\lbrace o_1,o_2,\ldots,o_T\rbrace$，求给定观测序列条件下，最可能出现的对应的状态序列，这个问题的求解需要用到基于动态规划的维特比算法。这个问题是HMM模型三个问题中复杂度居中的算法。

模型原理

pLSA与LDA差别及要点

  • pLSA是频率学派的，LDA是贝叶斯学派的，LDA为pLSA的两个参数$p(\omega|z)$和$p(z|d)$加了先验分布，即Dirichlet分布。
  • pLSA跟LDA的本质区别就在于它们去估计未知参数所采用的思想不同。前者是EM算法，通过$\phi_{k,j}$和$\theta_{i,k}$算出隐参数$z$的分布$p(z|d,\omega)$，然后通过最大化对数似然函数更新。后者是Gibbs采样，用条件分布的的抽样近似联合分布。

  • 最大后验MAP是由贝叶斯估计推导而来，比最大似然多乘了参数分布。

  • LDA假设文章之间是独立的，词之间是独立的，是词袋模型。

预备知识

gamma函数是阶乘在实数上的推广

多项分布是二项分布拓展到多维的情况

Beta分布和Dirichlet分布是概率的概率分布，是概率密度函数。

Dirichlet分布是Beta分布拓展到多维的情况。

Beta分布和Dirichlet分布的数学期望

Beta分布是二项分布的共轭先验分布；Dirichlet分布是多项分布的共轭先验分布

EM算法与变分推断

  • EM算法和变分推断都类似极大后验估计(MAP)或贝叶斯估计，区别是前者有隐含数据，变分推断是EM算法的扩展

  • EM算法的隐含数据由固定参数生成；变分推断隐含数据由随机变量参数生成

  • EM算法猜测隐含数据值并迭代；变分推断猜测隐含数据概率分布并迭代

  • EM算法可以推导出$Q(Z)$等于$P(Z|X)$；变分推断用$Q(Z)$逼近$P(Z|X)$

Gibbs采样与变分推断

  • Gibbs采样通过$P(Z_i|Z_{\neg i})$抽样；变分推断通过$Q^{[k]}(Z_i|X)\propto\exp\int_{Z^*_{\neg i}}Q^{[k]}(Z_{\neg i}|X)\log P(Z,X)dZ_{\neg i}$迭代

  • Gibbs采样通过抽取大量的样本估计真实的后验分布；变分推断限制近似分布的类型，从而得到一种局部最优，但具有确定解的近似后验分布

算法

  1. 初始化$Q^{(0)}(Z_i)$，可随机取
  2. 在第$k$步，计算$Z_{\neg i}$的边缘密度$Q^{[k]}(Z_{\neg i}|X)\propto\exp\int_{Z^*_i}Q^{[k-1]}(Z_i|X)\log P(Z,X)dZ_i$
  3. 计算$Z_{i}$的边缘密度$Q^{[k]}(Z_i|X)\propto\exp\int_{Z^*_{\neg i}}Q^{[k]}(Z_{\neg i}|X)\log P(Z,X)dZ_{\neg i}$
  4. 理论上$Q^{[\infty]}(Z_i|X)$将会收敛，则反复执行2和3直到$Q(Z_i)$，$Q(Z_{\neg i})$稳定，或稳定在某个小范围内
  5. 最后得$Q(Z)=Q(Z_i|X)Q(Z_{\neg i}|X)$

  • 我们输入的有$m$个文本，每个文本有$n$个词。而$A_{ij}$则对应第$i$个文本的第$j$个词的特征值，这里最常用的是基于预处理后的标准化TF-IDF值。$k$是我们假设的主题数，一般要比文本数少。SVD分解后，$U_{il}$对应第$i$个文本和第$l$个主题的相关度。$V_{jm}$对应第$j$个词和第$m$个词义的相关度。$\Sigma_{lm}$对应第$l$个主题和第$m$个词义的相关度。 则文本a和文本b的主题相似度可以用$U$的a行和b行向量余弦相似度计算。$U$相当于Doc2Vec，$V$相当于Word2Vec。

NLP-02-TextRank

发表于 2020-09-15 更新于 2020-09-16 分类于 自然语言处理

模型原理

PageRank

一个网页重要性是各入链网页重要性投票之和

其中，$V_i$表示某个网页，$V_j$表示链接到$V_i$的网页（即$V_i$的入链），$S(V_i)$表示网页$V_i$的PR值，$In(V_i)$表示网页$V_i$的所有入链的集合，$\lvert Out(V_j)\rvert$表示入链网页的出链个数，$d$表示阻尼系数，确保“孤岛”也有PR值。

TextRank

关键字提取

与PageRank类似，投票是共现频率

其中，$V_i$表示某个词语，$V_j$表示和$V_i$共现的词，$WS(V_i)$表示词语$V_i$的重要程度，$In(V_i)$表示词语$V_i$的所有共现词语集合，$\lvert Out(V_j)\rvert$表示共现词语和其他词共现的个数，$d$表示阻尼系数，确保“孤岛”也有WS值。权重项$\omega_{ji}$表示两个词语之间的边链接有不同的重要程度。

关键句提取

将文本中的句子看作节点

公式中，$S_i$，$S_j$分别表示两个句子，$w_k$表示句子中的词，那么分子部分的意思是同时出现在两个句子中的同一个词的个数，分母是对句子中词的个数求对数之和。分母这样设计可以遏制较长的句子在相似度计算上的优势。

算法

  • 把给定的文本T按照完整句子进行分割
  • 对于每个句子，进行分词和词性标注处理，并过滤掉停用词，只保留指定词性的单词，如名词、动词、形容词
  • 构建候选关键词图$G = (V,E)$，其中$V$为节点集，由上一步生成的候选关键词组成，然后采用共现关系（co-occurrence）构造任两点之间的边，两个节点之间存在边仅当它们对应的词汇在长度为K的窗口中共现，K表示窗口大小，即最多共现K个单词
  • 根据上面公式，迭代传播各节点的权重，直至收敛
  • 对节点权重进行倒序排序，从而得到最重要的T个单词，作为候选关键词
  • 由上一步得到最重要的T个单词，在原始文本中进行标记，若形成相邻词组，则组合成多词关键词

我们可以根据句子相似度公式循环计算任意两个句子之间的相似度，根据阈值去掉两个节点之间相似度较低的边连接，构建出节点连接图，然后计算TextRank值，最后对所有TextRank值排序，选出TextRank值最高的几个词语对应的句子作为摘要。

自由程度指标：

凝聚程度

互信息 (MI) 表示知道了随机变量$X$之后对另一个随机变量$Y$不确定性的减少

但由于这里随机变量的取值唯一，我们使用点间互信息(point-wise mutual information)作为衡量这一特征的指标

合词

从左到右依次计算相邻文本片段的凝聚程度，若达到设定的阈值，则将这两段文本片段整合为一块新的文本片段，然后继续去计算该整合后的文本片段与下一段相邻文本片段的凝聚程度。不断循环，直到无文本片段。

切词

从左到右依次计算相邻文本片段的凝聚程度（与合词不同，每次只计算相邻两个文本片段的凝聚程度），若未达到设定的阈值，则将这两段文本片段进行切分。不断循环，直到无文本片段。

算法

  1. 首先将文本拆分为文本片段（字符、英文单词等）
  2. 基于凝聚程度和切词方式对文本进行切分，获得一个粗的词库，同时也对文本做了一次粗的分词
  3. 基于自由程度从上一步的到得词库中筛选出词语
